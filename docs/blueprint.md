# **App Name**: EduBot Central

## Core Features:

- Category Display: Display the three user categories: Student, Class Representative, and Senior.
- Clickable Cards: Make each category a clickable card that directs the user to a category-specific help section.
- AI-Powered Recommendations Tool: Provide category-specific help via AI, using a tool that allows the LLM to selectively include facts based on user queries.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a trustworthy and educational feel.
- Background color: Very light Indigo (#E8EAF6).
- Accent color: Teal (#009688) to highlight interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern and clean look.
- Center the content on the page and use responsive design for different screen sizes.
- Use simple, educational icons for each category to enhance visual appeal.
- Add subtle hover animations on the cards to indicate clickability.
